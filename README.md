# TableplopCharacterSheets
A collection of character sheets (JSON format) and character sheet manipulation tools for the Tableplop Virtual Tabletop (VTT). 
Files ending in V01_xxx.json are for Version 1 of the Tableplop JSON character sheet specification, which is no longer used. 
Files ending in V02_xxx are for the "new" JSON specification. 
Tools are not yet ready for public use.
